Welcome! This is the first post on my new blog. I built this site using **Jekyll**, a static site generator that turns Markdown into beautiful web pages.

<!--more-->

## Why I Started This Blog

I wanted a simple, fast, and reliable place to share my thoughts, projects, and tutorials. With Jekyll and GitHub Pages, I can write posts in Markdown and publish them with a single `git push`.

## What You'll Find Here

- **Tutorials** — Step-by-step guides on various topics
- **Project Updates** — Progress on my current projects
- **Thoughts & Ideas** — Random musings and reflections

## How This Blog Works

This blog is fully static — no database, no server-side code, no JavaScript required. Every page is pre-built by Jekyll and served as plain HTML and CSS.

The design is responsive, so it looks great on any device. Posts are written in Markdown files inside the `_posts` folder, and each post automatically gets:

- A beautiful card preview on the homepage
- An auto-generated excerpt (truncated after `<!--more-->`)
- Previous/Next navigation between posts
- Category tags for organization
